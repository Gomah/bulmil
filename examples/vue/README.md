# @bulmil/vue-example

This example was bootstrapped with [Vue CLI](https://cli.vuejs.org/), and demonstrates how to setup
Bulmil with Vue via the `@bulmil/vue` package. See [`src/App.vue`](./src/App.vue).

## Usage

```bash
$: git clone https://github.com/gomah/bulmil

$: cd bulmil/examples/vue

$: npm install

$: npm run serve
```
